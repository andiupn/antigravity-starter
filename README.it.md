## 💡 Il problema: perché gli spazi di lavoro tradizionali falliscono nell'intelligenza artificiale
Gli agenti IA (Cursor, Gemini, Claude) sono straordinariamente capaci. Ma quando vengono inseriti in directory standard e caotiche, si perdono. Leggono cartelle irrilevanti, consumano il budget simbolico, hanno allucinazioni e inseriscono file nelle directory sbagliate perché non ci sono confini chiari.

---

## ⚡ La soluzione: tre pilastri dell'impalcatura AI premium

### 1. 🛰️ Visione perfetta per l'IA (AI-First Layout)
Abbiamo progettato ogni cartella, configurazione e regola `.gitignore` in modo che fungano da roadmap per la tua intelligenza artificiale. L'area di lavoro è estremamente pulita e consente a Cursor o Gemini di comprendere l'intera architettura **Next.js 16 + React 19 + Tailwind v4** in meno di 3 secondi. Zero gettoni sprecati, massima precisione.

### 2. 🤖 Team specializzato, non un chatbot generico
Non parli con una casella AI vuota. Ottieni un team di 2 agenti IA preconfigurati, altamente specializzati e autonomi:
- **`@code-reviewer`**: il tuo team di QA rigoroso e severo che esamina la qualità del codice, i lanugine e la struttura prima dei commit.
- **`@research-assistant`**: un agente di ricerca ad alta velocità ed economico, creato su misura per esplorare in sicurezza nuovi documenti e librerie API.

### 3. ⚡ Competenze con un clic (flussi di lavoro)
Esegui flussi di lavoro di sviluppo complessi con comandi semplici. Dotato di 8 flussi di lavoro di automazione riutilizzabili (`/skills`) che trasformano l'impalcatura manuale, il controllo dell'ambiente e la scrittura di test in un'attività di 1 secondo.

## 📂 Progetto dello spazio di lavoro

```
your-workspace/
  .gemini/             # AI agent configs, automation skills, and wiki
  docs/                # Pristine research and implementation plans
  src/                 # Main directory for your application source code
  .env.example         # Template for environment variables
  .gitignore           # Standard Git ignore file
  .mcp.json            # Model Context Protocol (MCP) configuration
  GEMINI.md            # AI coding rules & workspace guidelines
  LICENSE              # Project license (MIT License)
```

## 🚀 Inizia in 3 passaggi

### 1. Copia configurazioni:

```bash
cp .env.example .env
```

*(Inserisci il tuo `GITHUB_TOKEN` per abilitare le integrazioni del repository).*

### 2. Imposta le autorizzazioni AI:

__CODICE_BLOCCO_2__

*(Decommenta e imposta `permissions` in base al tuo sistema operativo).*

### 3. Inizia a scrivere codice:
Apri questo spazio di lavoro nel tuo editor compatibile con l'intelligenza artificiale preferito e sperimenta la velocità.
