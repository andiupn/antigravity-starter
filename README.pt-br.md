## 💡 O problema: por que os espaços de trabalho tradicionais falham na IA
Agentes de IA (Cursor, Gemini, Claude) são fenomenalmente capazes. Mas quando colocados em diretórios padrão e desordenados, eles se perdem. Eles leem pastas irrelevantes, estouram seu orçamento de tokens, têm alucinações e colocam arquivos nos diretórios errados porque não há limites claros.

---

## ⚡ A solução: três pilares do andaime de IA premium

### 1. 🛰️ Visão perfeita para IA (AI-First Layout)
Projetamos cada pasta, configuração e regra `.gitignore` para funcionar como roteiros para sua IA. O espaço de trabalho é extremamente limpo, permitindo que Cursor ou Gemini entendam toda a sua arquitetura **Next.js 16 + React 19 + Tailwind v4** em menos de 3 segundos. Zero tokens desperdiçados, precisão máxima.

### 2. 🤖 Equipe especializada, não um chatbot genérico
Você não fala com uma caixa de IA em branco. Você obtém uma equipe de 2 agentes de IA pré-configurados, altamente especializados e autônomos:
- **`@code-reviewer`** — Sua equipe de controle de qualidade rigorosa e rigorosa que analisa a qualidade, os lints e a estrutura do código antes de confirmar.
- **`@research-assistant`** — Um agente de pesquisa econômico e de alta velocidade, feito sob medida para explorar novos documentos e bibliotecas de API com segurança.

### 3. ⚡ Habilidades com um clique (fluxos de trabalho)
Execute fluxos de trabalho de desenvolvimento complexos com comandos simples. Equipado com 8 fluxos de trabalho de automação reutilizáveis ​​(`/skills`) que transformam o andaime manual, a verificação do ambiente e a gravação de testes em uma tarefa de 1 segundo.

## 📂 Projeto do espaço de trabalho

```
your-workspace/
  .gemini/             # AI agent configs, automation skills, and wiki
  docs/                # Pristine research and implementation plans
  src/                 # Main directory for your application source code
  .env.example         # Template for environment variables
  .gitignore           # Standard Git ignore file
  .mcp.json            # Model Context Protocol (MCP) configuration
  GEMINI.md            # AI coding rules & workspace guidelines
  LICENSE              # Project license (MIT License)
```

---

## 🚀 Comece em 3 etapas

### 1. Copiar configurações:

```bash
cp .env.example .env
```

*(Insira seu `GITHUB_TOKEN` para habilitar integrações de repositório).*

### 2. Configure as permissões de IA:

```bash
cp .gemini/settings.local.example.json .gemini/settings.local.json
```

*(Retire o comentário e defina `permissions` de acordo com seu sistema operacional).*

### 3. Comece a codificar:
Abra este espaço de trabalho em seu editor compatível com IA favorito e experimente a velocidade.
