## 💡 Sorun: Geleneksel Çalışma Alanları Neden Yapay Zeka'da Başarısız Oluyor?
AI Ajanları (İmleç, Gemini, Claude) olağanüstü yeteneklidir. Ancak standart, karmaşık dizinlere düştüklerinde kaybolurlar. İlgisiz klasörleri okurlar, token bütçenizi tüketirler, halüsinasyon görürler ve net sınırlar olmadığı için dosyaları yanlış dizinlere yerleştirirler.

---

## ⚡ Çözüm: Birinci Sınıf Yapay Zekalı İskelenin Üç Temeli

### 1. 🛰️ Yapay Zeka için Mükemmel Görüş (Yapay Zeka İlk Düzeni)
Her klasörü, yapılandırmayı ve `.gitignore` kuralını yapay zekanız için yol haritası görevi görecek şekilde tasarladık. Çalışma alanı son derece temiz olup Cursor veya Gemini'nin **Next.js 16 + React 19 + Tailwind v4** mimarinizin tamamını 3 saniyeden kısa sürede anlamasına olanak tanır. Sıfır jeton israfı, maksimum doğruluk.

### 2. 🤖 Genel bir Chatbot Değil, Uzman Ekip
Boş bir AI kutusuyla konuşamazsınız. Önceden yapılandırılmış, son derece uzmanlaşmış ve özerk 2 yapay zeka aracısından oluşan bir ekibe sahip olursunuz:
- **`@code-reviewer`** — Onaylamadan önce kod kalitesini, tüyleri ve yapıyı inceleyen titiz, katı QA ekibiniz.
- **`@research-assistant`** — Yeni API belgelerini ve kitaplıklarını güvenli bir şekilde keşfetmek için tasarlanmış yüksek hızlı, uygun maliyetli bir araştırma aracısı.

### 3. ⚡ Tek Tıklama Becerileri (İş Akışları)
Basit komutlarla karmaşık geliştirme iş akışlarını yürütün. Manuel iskele oluşturmayı, ortam kontrolünü ve test yazmayı 1 saniyelik bir göreve dönüştüren 8 yeniden kullanılabilir otomasyon iş akışı (`/skills`) ile donatılmıştır.

## 📂 Çalışma Alanı Planı

```
your-workspace/
  .gemini/             # AI agent configs, automation skills, and wiki
  docs/                # Pristine research and implementation plans
  src/                 # Main directory for your application source code
  .env.example         # Template for environment variables
  .gitignore           # Standard Git ignore file
  .mcp.json            # Model Context Protocol (MCP) configuration
  GEMINI.md            # AI coding rules & workspace guidelines
  LICENSE              # Project license (MIT License)
```

## 🚀 3 Adımda Başlayın

### 1. Yapılandırmaları kopyalayın:

```bash
cp .env.example .env
```

*(Depo entegrasyonlarını etkinleştirmek için `GITHUB_TOKEN` kodunuzu ekleyin).*

### 2. AI izinlerini ayarlayın:

```bash
cp .gemini/settings.local.example.json .gemini/settings.local.json
```

*(Açıklamaları kaldırın ve `permissions`'i işletim sisteminize göre ayarlayın).*

### 3. Kodlamaya Başlayın:
Bu çalışma alanını favori yapay zeka dostu düzenleyicinizde açın ve hızı deneyimleyin.
