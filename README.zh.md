## 💡 问题：为什么传统工作空间无法实现人工智能
人工智能代理（Cursor、Gemini、Claude）能力非凡。但当放入标准的、杂乱的目录中时，它们就会迷失方向。他们读取不相关的文件夹，耗尽你的代币预算，产生幻觉，并将文件放置在错误的目录中，因为没有明确的界限。

---

## ⚡ 解决方案：高级人工智能脚手架的三大支柱

### 1. 🛰️ AI 完美愿景（AI 优先布局）
我们设计了每个文件夹、配置和 `.gitignore` 规则来充当 AI 的路线图。工作空间非常干净，让 Cursor 或 Gemini 在不到 3 秒的时间内了解整个 **Next.js 16 + React 19 + Tailwind v4** 架构。零浪费代币，最大准确性。

### 2. 🤖 专业团队，而不是通用聊天机器人
你不会和一个空白的人工智能盒子说话。您将获得一支由 2 个预先配置的、高度专业化且自主的 AI 代理组成的团队：
- **`@code-reviewer`** — 您严谨、严格的 QA 团队，在提交之前检查代码质量、lints 和结构。
- **`@research-assistant`** — 高速、经济高效的研究代理，专为安全探索新的 API 文档和库而定制。

### 3. ⚡ 一键式技能（工作流程）
使用简单的命令执行复杂的开发工作流程。配备 8 个可重复使用的自动化工作流程 (`/skills`)，将手动搭建、环境检查和测试编写变成 1 秒任务。
