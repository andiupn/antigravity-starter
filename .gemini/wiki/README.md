# Wiki — Workspace Knowledge Base

Repositori dokumentasi dan basis pengetahuan proyek.

> **Catatan:** Wiki ini sengaja di-exclude dari scan otomatis AI Agent (dikonfigurasi di `.geminiignore`) agar tidak membanjiri jendela konteks obrolan.
> Akses wiki secara manual menggunakan perintah `Read` jika memerlukan entri referensi spesifik.

---

## Struktur Wiki saat ini

```
wiki/
  README.md
  patterns/
    web-tech-2026.md     ← Standar teknologi web 2026
```

---

## Cara Mengisi

1. Buat file `.md` baru di subdirektori yang sesuai (misal di bawah `patterns/`).
2. Gunakan format penamaan berkas `kebab-case` yang deskriptif.
3. Jalankan `/wiki-lint` untuk memeriksa keutuhan tautan antar-halaman.
