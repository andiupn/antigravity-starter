## 💡 Problemet: Varför traditionella arbetsytor misslyckas med AI
AI-agenter (Cursor, Gemini, Claude) är fenomenalt kapabla. Men när de hamnar i vanliga, röriga kataloger, går de vilse. De läser irrelevanta mappar, blåser igenom din tokenbudget, hallucinerar och placerar filer i fel kataloger eftersom det inte finns några tydliga gränser.

---

## ⚡ Lösningen: Tre pelare av premium AI-ställningar

### 1. 🛰️ Perfect Vision för AI (AI-First Layout)
Vi designade varje mapp, konfiguration och `.gitignore`-regel för att fungera som färdplaner för din AI. Arbetsytan är extremt ren, vilket gör att Cursor eller Gemini kan förstå hela din **Next.js 16 + React 19 + Tailwind v4**-arkitektur på mindre än 3 sekunder. Noll bortkastade tokens, maximal noggrannhet.

### 2. 🤖 Specialiserat team, inte en generisk chatbot
Du pratar inte med en tom AI-box. Du får ett team av 2 förkonfigurerade, högt specialiserade och autonoma AI-agenter:
- **`@code-reviewer`** — Ditt rigorösa, strikta QA-team som granskar kodkvalitet, ludd och struktur innan du åtar dig.
- **`@research-assistant`** — En höghastighets, kostnadseffektiv forskningsagent som är skräddarsydd för att utforska nya API-dokument och bibliotek på ett säkert sätt.

### 3. ⚡ One-Click Skills (arbetsflöden)
Utför komplexa utvecklingsarbetsflöden med enkla kommandon. Utrustad med 8 återanvändbara automatiseringsarbetsflöden (`/skills`) som förvandlar manuell byggnadsställning, miljökontroll och testskrivning till en uppgift på 1 sekund.

## 📂 Arbetsyta Blueprint

```
your-workspace/
  .gemini/             # AI agent configs, automation skills, and wiki
  docs/                # Pristine research and implementation plans
  src/                 # Main directory for your application source code
  .env.example         # Template for environment variables
  .gitignore           # Standard Git ignore file
  .mcp.json            # Model Context Protocol (MCP) configuration
  GEMINI.md            # AI coding rules & workspace guidelines
  LICENSE              # Project license (MIT License)
```

## 🚀 Kom igång i 3 steg

### 1. Kopiera konfigurationer:

```bash
cp .env.example .env
```

*(Infoga din `GITHUB_TOKEN` för att aktivera repository integrations).*

### 2. Ställ in AI-behörigheter:

```bash
cp .gemini/settings.local.example.json .gemini/settings.local.json
```

*(Avkommentera och ställ in `permissions` enligt ditt operativsystem).*

### 3. Börja koda:
Öppna den här arbetsytan i din AI-vänliga favoritredigerare och upplev hastigheten.
